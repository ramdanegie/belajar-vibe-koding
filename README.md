## Getting Started

```bash
# Install Bun
curl -fsSL https://bun.sh/install | bash

# Clone
git clone https://github.com/ramdanegie/belajar-vibe-koding.git
cd belajar-vibe-koding

# Run a script
bun run index.ts
```

## Why Bun?

Bun is a practical drop-in replacement for Node.js, so you can run existing Node.js applications with Bun and gain access to its native APIs. Bun is designed as a drop-in replacement for Node.js, offering:
- 4x faster startup
- Built-in TypeScript support
- Native bundler and transpiler
- Complete Node.js API compatibility
